Git Commands

1) Inicijalizacija ve? postoje?eg gita:

git clone [Full HTTPS adresa projekta] [odredišna putanja]

Primjer: 
git clone https://gitlab.com/dugonogi-progi/radno_vrijeme.git 
Objašnjenje naredbe: preuzmi kopiju projekta i spremi u trenutni aktivni direktorij (jer nije naveden opcionalni dio, odredišna putanja).

2) Osvježavanje trenutnog stanja projekta:

git pull origin [OPCIONALNO: naziv_brancha]
OPCIONALNO: ukoliko želimo osvježiti branch umjesto maina

3) Priprema za spremanje u lokalni repozitorij:

git add * // pripremanje svih izmijenjenih datoteka
git add filename.ext // dodavanje pojedina?ne datoteke

4) Provjera trenutno aktivnog brancha i spremanja  

git status
Izbacit ?e nam u kojem smo branchu, koje datoteke smo izmijenili, a nismo commitali te datoteke koje smo commitali.

5) Spremanje u lokalni repozitorij:

git commit -m "Kratka poruka o novim promjenama"

6) Spremanje na GitLab repozitorij:

git push origin branch
Primjer: 	git push origin main
		git push origin dev
		git push origin doc

7) Prebacivanje u ve? postoje?u granu:

git checkout grana
Primjer: 	git checkout main
		git checkout doc

8) Kreiranje nove grane:

git branch naziv_grane
Objašnjenje: kreiraj granu s nazivom "naziv_grane"
Dodatno: -b zastavica ?e se po završetku naredbe prebaciti u novokreiranu granu

Primjer: 	git branch novaGrana [-b]

9) Spajanje grana:

git merge grana

Objašnjenje: Iz navedene grane preuzima sve datoteke u aktivnu granu.
Primjer: pretpostavi da je aktivna grana main.
git merge doc -> iz doc grane ce preuzeti sve datoteke u main granu.
